服务端渲染

  • 为什么要服务端渲染？

    在 SPA 模式下，所有的数据请求和 Dom 渲染都在浏览器端完成，所以当我们第一次访问页面的时候很可能会存在“白屏”等待。

  1. 首屏等待：

    而服务端渲染所有数据请求和 html内容已在服务端处理完成，浏览器收到的是完整的 html 内容，可以更快的看到渲染内容 。

  2. SEO

    SPA 模式对搜索引擎不够友好

  • 方案：

    第一次访问页面是服务端渲染，基于第一次访问后续的交互就是 SPA 的效果和体验，还不影响SEO 效果，这就有点完美。
