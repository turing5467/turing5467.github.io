模块化开发(JS)

Aug 23, 2019

模块化开发

  1. CommonJS
  2. ES6Module
  3. AMD

CommonJS

  • 只能在Node.js环境下运行

  • 导入require()

    • 对于不同文件的扩展名，载入方法也有所不同

      • .js 通过fs模块同步读取后编译执行
      • .node 通过C/C++编写的扩展文件，通过dlopen()加载编译生成的文件
      • .json 通过fs模块同步读取后，用JSON.parse()解析返回结果
      • 其余都被当做.js文件载入

      每一个编译成功的模块都会将其文件路径作为索引缓存在Module._cache对象上，以便提高二次引入的性能

  • 导出module.exports = xx

ES6

  • 只能在浏览器端运行

  • 导入：import xx from 'xx'

  • 导出：export {a}

    默认导出：export default {xx} | xx
