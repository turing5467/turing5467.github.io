高阶函数

高阶函数是指至少满足下列条件之一的函数

  1. 函数可以作为参数被传递
  2. 函数可以作为返回值输出
  • 应用
    1. 函数柯里化
    2. AOP
    3. 函数防抖、节流
    4. 分时函数

currying(柯里化)

  • 在一个函数中，首先填充几个参数，然后返回一个新的函数的技术，成为函数的柯里化

  • 通常可用于在不侵入函数的前提下，为函数预置通用参数，供多次重复调用

实现

  • 原理：用闭包把传入参数保存起来，当传入参数的数量足够执行函数时，就开始执行函数

aop

AOP(面向切面编程)的主要作用是把一些和核心业务逻辑模块无关的功能抽取出来，然后再通过“动态织入”的方式掺到业务模块种。这些功能一般包括日志统计,安全控制,异常处理等。AOP是Java Spring架构的核心。

惰性加载函数

由于浏览器的差异性，常常要做各种兼容，一般是通过条件判断来执行分支语句。

缺点：每次执行都会执行if条件分支，虽然开销不大，但明显是多余的。

改进：一旦进入分支，修改函数的实现，这样下一次调用的就是我们期望的函数，而无需再次进入条件指针。
