Hexo搭建个人博客

Sep 2, 2019

Hexo

  1. 安装hexo $ cnpm install hexo -g

搭建博客

  • 主要流程
    ❏ 在本地目录中使用hexo来初始化博客仓库。
    ❏ 在Github网站使用自己的账号来创建对应的托管仓库。
    ❏ 对本地的hexo博客项目进行配置和Github官网的托管仓库对接。
    ❏ 在命令行中通过hexo命令来生成博客、预览博客和发布博客到远程仓库。

  • 选择主题themes

1-初始化

  1. 进入自己创建的blog文件夹
  2. hexo init 初始化
  3. 编写文章

2-托管到github

  1. 新建一个仓库命名为用户名.github.io(复制一下https地址)
  2. 安装一个模块 cnpm i --save hexo-deployer-git
  3. 配置根目录下的_config.yml(注：每个:后必须有空格)
  1. hexo d 对接到远程仓库

hexo基础命令

  • hexo n [layout] <title>
  • hexo g 生成博客文件
  • hexo s 开启hexo服务器(访问localhost:4000)
  • hexo server -p [端口号] 配置端口号

目录结构

更换主题

主题地址

  • 目录: d:\\myBlog
  1. 找到对应仓库，git clone xx themes/主题名 
    按照其说明文档，可能需要安装其它依赖
  2. 修改根目录下的_config.yml的theme字段为主题名称
  3. hexo clean
  4. hexo g
  5. hexo d同步到github仓库

更新文章后

  1. 切换至博客文件夹

  2. hexo clean

  3. hexo g

  4. hexo d同步到github仓库
